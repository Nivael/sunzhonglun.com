石头 / 石头把我的自行车磕得双轮爆胎 / 在我，做出一个简单的飞越动作时 / 这也说明了 / 我爱骑的品质 / 因此我将它遗弃在公司的 / 泊车场 / 一个多月，快两个 / 拿，北京的风沙 / 羞辱它 / 就像这风沙每天早晨，羞辱我 / 一样 / 可风沙忽然，停了 / 以太的饱和度忽然，变得很高 / 许多树叶变得，比事实上更层次分明 / 不管怎样，我 / 还是在和它斗气，我让它瘫痪着反正 / 它也没法自己修理自己 / 然后正如，我一向擅长的 / 移情别恋 / 所以这篇 / 跟它没关系，它仅仅 / 贡献了两个垃圾轮子，和一个 / 跟石头分享的开头。

我要说的 / 那个新情人，蓝色的 / 复制品，伴我每天早上 / （不得不）缓慢，经过那些 / 高饱和度的街道 / 有惊无险地闯过，两个红灯 / 反正，每个人都闯 / 这让 / 绿灯都没人了，显得 / 很危险。/ 它和斯皮尔伯格电影里那匹 / 磕了药的战马，一样 / 光荣地，随时散架地伴我 / 抵达终点 / 就像舞池里那些/ 能在各种关节摇动的 / 达人，甚至 / 更厉害，因为它所有关节都在 / 抖动，让我的腿 / 也抖动起来 / 就连舞池里的达人，也无法 / 使我做到。/ 我该如何回答 / 当你的蓝色主人问我 / 故障部位 / 并且，最多勾选三项。

我可以选你 / 也满可以选 / 你的死对头，黄色的 / 复制品 / 你们像 / 我爱我家和链家的西装人一样 / 不对付，常常准备 / 斗殴 / 但大部分时候，分而治之，车头们摆向 / 两个方向，体现 / 这个国家罕有的 / 对消极自由的尊重。/ 我选你，出于朴素的 / 理由 / 因为两年前我购买了 / 你蓝色主人的骑行卡 / 一直 / 忘记取消 / 所以，现在骑你 / 两小时内0元。/ 你的蓝色主人 / 靠坑我这样的 / 傻逼 / 发了大财 / 最近还换了 / 更精明的老板，让我对 / 自己的命运 / 隐隐发愁。

我一直好奇 / 你眼中的世界是怎样。/ 正如每天早晨经过 / 那些分形般规整的树叶，我也 / 好奇每一片，看到的/ 是否只有微妙视角的差别。/ 其实人，也没有 / 什么了不起 / 也是 / 有丝分裂的产品 / 这庞大复制品世界中的 / 一环，如我们 / 穿的衣物，吃的汉堡，和 / 声称古法秘方的外卖 / 我们也受 / 眼睛，身体，心灵的 / 囚禁。/ 我们响亮的 / 对独特性的呼嚎 / 对理型的 / 觊觎 / 恐怕要，让他发笑。/ 不过话说回来 / 我们为何，要像他一样 / 同时看到所有的地方，这何尝 / 对人来说 / 不是一种剥夺。

然而 / 有一种命运，人 / 似乎可以规避 / 而你，蓝色的复制品情人 / 不行。/ 我们行走留下脚印 / 脚印不会成为我们的负担 / 却会成为你的。/ 你在世上行走、展示得愈多 / 越增加你被复制的概率 / 就像光芒 / 它越让人向往，渴望，匍匐 / 一旦落入手中 / 灯泡亮起 / 便有了拙劣的命运。/ 因此，蓝色的复制品情人 / 你似乎得等到被淘汰的那天 / 才能停止自己不断贬值的命运。/ 那些贴在你身上 / 代开发票和，1:1真人硅胶娃娃体验馆的 / 二维码 / 通向一个又一个复制品空间的 / 通道 / 将成为你的第一张名片，血管，骨骼 / 北京风沙那么大 / 化石也许 / 很容易形成 / 城市的考古学家将会发现 / 化石上凸起的那块 / 骨头 / 你将戳破 / 另外许多轮胎。
